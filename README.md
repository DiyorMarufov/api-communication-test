# API Communication Test

A Node.js project demonstrating HTTP communication with a test API, receiving a two-part code, and combining it to retrieve a final message.  

---

## üöÄ Overview

This project shows how to:

1. Send a `POST` request to the test API with your message and webhook URL.  
2. Receive the **first part** of the code from the API response.  
3. Receive the **second part** of the code at your webhook URL.  
4. Concatenate both parts and send a `GET` request to retrieve the final message.  

**Tech used:**  
- Node.js  
- Express  
- Axios  
- Localtunnel (for exposing your local webhook to the internet)
